readTimetable(callback)

Asynchronously read the timetable from the database file.
Parameters:
Name Type Description
callback function code to be executed after database loads. Takes parameters (String err, Timetable timetable). If an error occurs while loading the database, then err will contain the error and timetable will be null. If the database loads successfully, then err will be null and timetable will contain the timetable.
